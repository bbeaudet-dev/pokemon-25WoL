Manual target selection should eventually happen before round play begins.

The current implementation only creates one setup round at a time, so only that round's hintmaster can choose or confirm targets. For true manual mode, every player should pick and lock their own target list during a shared pre-game setup step.

That will likely require storing per-player pending target selections on the game, showing the picker to all players at once, and starting the first round only after everyone is locked in.
