## Trainer Names

Default guest names can use playful Pokemon-style combinations, such as:

- Hungry Munchlax
- Sleepy Snorlax
- Energized Pikachu
- Comfy Lugia

Later, lobby joins should check for duplicate display names and reroll or suffix the generated name when needed.
